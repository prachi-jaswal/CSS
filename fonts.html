Fonts

The different Css font properties are.

  • font-family
  • font-size
  • font-style
  • font-variant
  • font-weight
  • font

We can specify a list of font names for the text contained inside an HTML element, using font-family property. Ex:- p { font-family: Arial, serif, "Times New Roman"; } If the name of a font family is more than one word, it must be in quotation marks. Ex:- “Times New Roman” Times

We can specify a font size for the text. We can specify size in the form of percentage, px, em.

  1. xx-small
  2. x-small
  3. small
  4. medium
  5. large
  6. x-large
  7. xx-large
  8. smaller
  9. larger
